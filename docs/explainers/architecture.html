always-on add-ons
Always deployed
🔵
Cilium CNI (cilium.tf)
Deployed unconditionally — RKE2 is started with cni: none so Canal is disabled. VXLAN tunnel, MTU 1400 (Hetzner 1500 − 50 overhead), cluster-pool IPAM (/24 per node), kubeProxyReplacement="false" for CCM compatibility. Also applies a CiliumClusterwideNetworkPolicy blocking egress to 169.254.169.254 (Hetzner metadata API).
